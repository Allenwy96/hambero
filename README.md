## ğŸš€ Getting Started

### Viewing the Website

1. Open `index.html` in your web browser
2. Navigate through the different pages using the navigation menu

### Customizing Product Images

1. Navigate to the `images/` folder
2. Replace the placeholder images with your actual product photos
3. Keep the same filenames (product-1.jpg, product-2.jpg, etc.) or update the HTML files
4. See `images/README.md` for detailed instructions

### Customizing Content

- **Company Information**: Update company details in the footer and contact page
- **Products**: Edit `products.html` to add, remove, or modify products
- **Styling**: Modify `css/style.css` to change colors, fonts, and layout
- **Email Address**: Update email addresses throughout the site to match your domain

### Fonts

The website uses Inter font from Google Fonts. To change the font:
1. Update the Google Fonts link in the HTML files
2. Update the font-family in `css/style.css`
