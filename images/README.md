# Product Images

This folder contains product images for the Hambero website.

## How to Replace Product Images

To replace the placeholder images with your actual product photos:

1. Add your product images to this folder
2. Rename your images to match these exact filenames:
   - `product-1.jpg` - Premium Headphones
   - `product-2.jpg` - Wireless Mouse
   - `product-3.jpg` - USB-C Cable
   - `product-4.jpg` - Portable Charger
   - `product-5.jpg` - Bluetooth Speaker
   - `product-6.jpg` - Phone Stand
   - `product-7.jpg` - LED Desk Lamp
   - `product-8.jpg` - HD Webcam
   - `product-9.jpg` - Wireless Keyboard

3. The website will automatically display your images

## Image Specifications

For best results, use images with these specifications:
- **Format**: JPG, PNG, or WebP
- **Dimensions**: At least 800x600 pixels (4:3 aspect ratio recommended)
- **File Size**: Under 500KB for optimal loading speed
- **Quality**: High resolution but optimized for web

## Notes

- The placeholder.svg file will be used as a fallback if a product image is not found
- You can keep the same filenames or update the HTML files to match your preferred naming scheme
- All images should be web-optimized to ensure fast page loading
